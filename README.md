# CubeTrainer — OLL & PLL Mastery

A speedcubing trainer for learning and practising all 78 OLL and PLL algorithms.

**Try it now: [https://mpberk.github.io/Cube-Trainer/](https://mpberk.github.io/Cube-Trainer/)**

## Features

- **Learn Mode** — Study unlearned algorithms 5 at a time with diagrams
- **Quiz Mode** — Test recall on algorithms you've marked as learned
- **Library** — Browse, search, and filter all 78 algorithms
- **OLL Diagram Editor** — Set up OLL patterns by toggling stickers interactively
- **Progress Tracking** — Track learned algorithms with stats saved to local storage
